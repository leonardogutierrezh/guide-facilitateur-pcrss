# Tâche 4 — 7.2 Code de bonne conduite (mise en œuvre)

Tous les acteurs signent le code. **Code :** observer les lois en vigueur au Mali ; respecter les droits des parties prenantes ; respecter les exigences/étapes du DCC ; bonne ambiance de travail ; veiller à la participation des femmes/pauvres/vulnérables/religieux ; s'abstenir de corruption/fraude ; signaler fraude/corruption ; respecter cultures locales et lois coutumières ; suivre le manuel d'opération ; signaler tout problème au supérieur.
**À faire :** équité, honnêteté, respect ; éviter tout harcèlement (physique, moral, sexuel) ; respecter l'équité-genre ; règles d'hygiène et sécurité ; mesures de sauvegarde E&S ; permis pour les zones de conservation forestières ; respecter us et coutumes ; visite de courtoisie aux autorités locales/traditionnelles ; utiliser les messages clés lors des visites de site.
**À ne pas faire :** accepter pots-de-vin/ristournes/gratifications ; divulguer des informations confidentielles pour un gain ; falsifier des documents ; discriminer ; corruption ; injures/délits d'initiés ; pénétrer les lieux saints/sacrés/tabous sans autorisation coutumière ; fumer dans les lieux interdits ; photographier/filmer/enregistrer des personnes ou lieux sacrés sans autorisation.
