# Activité — Gestion environnementale et sociale pendant les travaux / E&S management during implementation

> Phase 5 · Activité 2 · Étape 11 · Source : Part 2, §7

## Quick reference (EN)

During works, activities can harm the environment and the beneficiary/neighbouring population. The project requires measures to avoid, reduce, mitigate and (where technically/financially possible) compensate damage. Awareness and application run throughout implementation.

**Risk types by phase:** site preparation (noise, traffic accidents, air pollution); construction (worker accidents, soil pollution from hydrocarbon spills); operation (operator accidents, odour nuisances from waste); throughout (GBV/SEA/SH risks). **Consequences of ignoring E&S policies:** serious economic/social/environmental harm; accidents/deaths; harm to sacred sites; complaints; delays; lower disbursement rates; reputational harm to PCRSS; project suspension.

**Before works, the contractor must:** insure the site and every employee; declare each employee to INPS; have a permanent **HSE specialist** on site for the site-level ESMP (PGES-C); hold daily 30-min site meetings (environment, health-safety, MGP, SEA/SH); sign and have each employee sign the code of conduct; avoid child labour and human trafficking; provide and enforce **PPE** (for employees and visitors); signpost and demarcate the site; inform/sensitize on the MGP, accident risks, transmissible diseases (STI-HIV/AIDS), SEA/SH/VCE; provide condoms on request; set up a grievance mechanism. **AR/FT/ERAR** prepare the specific safeguard instrument, validate the PGES-C, run sensitization and start-up meetings, enforce PPE, obtain administrative authorizations (tree-felling, building permits).

All actors sign the **code of conduct**. Nine E&S forms (**ES1–ES9**).
