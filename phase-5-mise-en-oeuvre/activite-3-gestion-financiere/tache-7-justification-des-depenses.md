# Tâche 7 — 8.7 Justification des dépenses

Les **pièces justificatives** attestent une autorisation/entrée/sortie de fonds : reçu, facture, bon de caisse, bon de commande/contrat, bordereau de livraison/PV de réception/attestation de service fait, caution de bonne exécution/retenue de garantie, décompte des travaux/rapports, facture définitive, copie du chèque acquitté, état de paiement. Éléments attendus : raison sociale et adresse du prestataire, contact, lieu ; nom du client ; objet ; montant ; date ; nom/prénom du fournisseur ; signature ; cachet ; acquittement ; décharge. Sources : entreprises, fournisseurs, prestataires, mairie. NB : transport taxi-moto justifié par documents internes (bon de caisse, reçu interne, décharge) approuvés par le Président.
