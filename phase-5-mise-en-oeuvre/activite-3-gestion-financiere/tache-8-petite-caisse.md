# Tâche 8 — 8.8 Petite caisse

Pour les dépenses non réglables par chèque (transport, etc.). **Plafond : 50 000 FCFA.** Approvisionner le jour des paiements pour limiter les risques. Le Trésorier est l'unique détenteur. Tous les mouvements enregistrés dans le **Cahier de caisse (F13)**. En fin de mois, état de dépenses mensuelles (**F14**). NB : depuis le 02/07/2021, factures normalisées avec **NIF** des prestataires/clients.
