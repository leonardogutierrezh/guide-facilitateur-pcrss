# Tâche 1 — 8.1 Rôles et responsabilités

Dans le DCC, la communauté reçoit directement les fonds et les gère. Le B-CDVFQ (surtout le trésorier, avec appui FC et assistant comptable régional) : tient les livres comptables ; garde les fonds et gère le compte bancaire ; aide à préparer les budgets ; effectue les paiements après autorisation ; prépare registres et rapports ; retourne tout solde sur le compte du village ; tient les livres pour vérification/inspection.
