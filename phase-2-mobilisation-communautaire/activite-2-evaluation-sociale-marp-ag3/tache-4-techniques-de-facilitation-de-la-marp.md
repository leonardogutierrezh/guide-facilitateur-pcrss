# Tâche 4 — 4.4 Techniques de facilitation de la MARP

Créer un environnement confortable ; établir les règles de base ; expliquer simplement dans la langue locale ; poser de bonnes questions ouvertes ; encourager les plus calmes et les femmes/marginalisés ; écouter et faire écouter ; reformuler ; ne pas prendre l'avis d'un seul pour celui du groupe (« Qu'en pensent les autres ? ») ; sonder ; enregistrer ce qui est dit.
