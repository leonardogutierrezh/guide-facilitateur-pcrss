# Tâche 9 — 4.9 Diagramme de Venn (étapes)

Préparation : cercles de papier de différentes tailles.
1. Introduction — lister les groupes/individus/organisations internes et externes importants, puis examiner leurs liens.
2. Liste des acteurs (dans le village : chef, ménages, B-CDV, anciens, chefs coutumiers/religieux, enseignants, groupes de femmes/agriculteurs/éleveurs/jeunes, protection sociale ; hors village : sous-préfecture, centre de santé, commerçants, banques, prestataires, STDE, mairie, ONG, FDS…).
3. Influence — noter les noms sur des cercles (grands = importants, petits = moins influents).
4. Création du diagramme — position/distance = contact/influence/relation (cercles séparés = pas de contact ; qui se touchent = échange d'info ; faible chevauchement = certaine coopération ; grand chevauchement = beaucoup de coopération ; longue distance = désaccord/conflit). Chevauchements remplaçables par des flèches.
5. Discussion : qui travaille sur quoi ; qui inclut/exclut les femmes ; rôle des chefs ; liens avec organismes externes ; soutiens en cas de chocs ; relations difficiles ; rôle pour mieux répondre aux risques.
6. Résumé sur Padex ; lier à la planification.
