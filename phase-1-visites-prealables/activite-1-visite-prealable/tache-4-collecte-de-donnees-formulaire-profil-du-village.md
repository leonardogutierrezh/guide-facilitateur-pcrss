# Tâche 4 — 1.4 Collecte de données — Formulaire Profil du village

La visite préalable sert aussi à commencer à recueillir des données secondaires. Le formulaire **Profil du village/quartier/fraction** dans l'application DCC doit être rempli lors de la visite préalable. Certaines informations sont utiles pour planifier la 1ère réunion.
