# Tâche 3 — 5.1 Acteurs — externes

| Acteur | Rôles |
|---|---|
| SC, FC | Appui-conseil (mobilisation, dynamique communautaire). **Ne fait pas partie** du dispositif de passation ; n'intervient en aucun cas comme acteur opérationnel |
| FT | Appuie le CDV à finaliser les DAOL (aspects techniques) ; assiste à l'évaluation (correction des prix) en qualité d'**observateur** ; assure le contrôle des travaux pour le compte du CDVFQ |
| Entrepreneur/Fournisseur/Consultant | Signe le contrat et réalise le sous-projet/la tâche |
| Mairie | Délègue la maîtrise d'ouvrage ; suit la passation sans se substituer ; **ne donne pas d'avis a priori** |
| STDE | Appui et conseil techniques pour l'exécution |
| AR, ERAR | Appui technique à toutes les étapes ; supervise via un plan de passation élaboré par le B-CDV |
