# Tâche 2 — Introduction à l'audit social

Réunion communautaire d'une demi-journée à la fin de chaque cycle d'investissement, dès que le sous-projet est achevé. Rassemble les organes du CDVFQ, la communauté et les autres parties prenantes. Les organes informent sur l'identification, la planification, la mise en œuvre, les résultats et le coût ; la communauté commente/demande des clarifications ; ensemble, ils formulent des recommandations pour le cycle suivant.
**Objectifs :** promouvoir la redevabilité ; promouvoir l'apprentissage ; faire des recommandations ; renforcer l'appropriation ; apprécier les résultats.
**Qui :** les organes du CDVFQ dirigent ; FC/FT assistent ; tous les membres invités (quorum : ≥ moitié des ménages, idéalement parité, effort particulier pour femmes, vulnérables, handicapés) ; parties prenantes (STDE, mairies, OSC/ONG, enseignants, santé) ; le chef de V/Q/F.
