# Tâche 13 — Compétences de facilitation pour l'audit social

Objectif : discussion efficace, accords clairs, participation active et égale (attention aux femmes et vulnérables). Une bonne facilitation : compréhension claire des objectifs ; agenda connu et suivi ; informations claires sans s'étendre ; les timides à l'aise ; vote bien expliqué ; discussions de groupe bien facilitées ; réunion qui ne s'éternise pas ; décisions claires acceptées par la majorité ; opinions fortes gérées sans conflit ouvert.
**Compétences-clés :** expliquer/réexpliquer le sujet ; poser de bonnes questions ; questions de suivi ; reformuler/répéter ; montrer intérêt et appréciation ; équilibrer la participation ; rechercher les points d'accord et les énoncer ; recommandations claires (qui fait quoi).
**Gestion des problèmes :** perte d'intérêt (pause/énergisant, réexpliquer, nouveau sujet) ; femmes/vulnérables exclus (insister sur la participation égale, demander leurs idées en premier) ; interruptions (intervenir poliment, rappeler les règles) ; dispute (faire écouter et reformuler, identifier points communs/divergences) ; section se plaignant de ne pas avoir bénéficié (enregistrer les plaintes, laisser le B-CDVFQ répondre) ; une personne domine (donner la parole aux autres, éviter le contact visuel) ; silence (encourager, petits groupes) ; présentateurs qui débordent (limite de temps, rappels polis, encourager à rester).
