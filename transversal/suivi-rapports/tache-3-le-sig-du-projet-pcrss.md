# Tâche 3 — 8.3 Le SIG du projet PCRSS

Le Système d'Information et de Gestion (SIG) est une plateforme intégrée centralisée qui assiste l'UEP et les partenaires : suivi des activités/progrès ; décisions éclairées ; signalement des problèmes ; mesures correctives ; évaluation de l'impact ; acquisition/contrats ; contrôle financier. Les formulaires doivent être saisis dans le SIG ou les applications DCC et MGP, en temps voulu (données saisies sur mobile, synchronisées avec le serveur UEP).

| Formulaires/outils | Sujet | Responsable |
|---|---|---|
| Application DCC | Profil du village | FC/FT |
| Application DCC | Calendrier du cycle du projet | FC/FT |
| Application DCC | Priorités de développement du village | FC/FT |
| Application MGP | Enregistrement et suivi des plaintes | FC/FT |
| GEMS | Suivi des activités des comités/CDV/prestataires, collecte des données, respect des normes techniques | FC/FT |

L'objectif du SIG est un flux d'information continu pour le suivi, les rapports et les décisions. Chaque facilitateur collecte les formulaires, les saisit sur tablette, puis synchronise. Le CCR vérifie avant téléversement sur le serveur national et signale les erreurs techniques à l'équipe SIG.
