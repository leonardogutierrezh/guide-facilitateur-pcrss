# Tâche 3 — 6.3 Critères d'éligibilité aux sous-projets

- L'enveloppe globale du village guide approximativement le nombre de sous-projets finançables.
- La subvention globale est allouée **par village** (pas par sous-projet).
- Le montant global des sous-projets ne peut dépasser l'allocation du village.
- Aucun sous-projet ne peut augmenter la vulnérabilité du village aux chocs climatiques.
- Les villages peuvent collaborer sur un sous-projet commun (gestion par consensus).
- Les sous-projets sont identifiés sur la base des priorités (3ème AG).
- Tout projet non admissible est documenté dans le PDC (autres sources).
