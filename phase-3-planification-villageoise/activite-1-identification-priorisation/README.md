# Activité 5 — Identification et priorisation des activités / Identifying & prioritizing development activities

> Phase 3 · Activité 1 · Étape 5 · Source : Part 1, §5

## Aperçu (FR)

Les prochaines étapes sont la partie la plus importante de l'évaluation participative : les communautés utilisent les résultats du diagnostic pour commencer à construire le PDC. Le facilitateur établit le lien entre l'évaluation des défis/besoins et l'élaboration du PDC.

## Quick reference (EN)

This is the most important part of the participatory needs assessment — the community turns the MARP results into the start of the **PDC**. The facilitator links the diagnosis to plan-building. Sequence: review MARP results in plenary → in focus groups, build a **problems/challenges list**, a **vision** (5 years), a **list of development activities/sub-projects**, then **prioritize in groups** → **consolidate and prioritize in plenary**.

**Prioritization criteria (explain before voting):** number of beneficiaries; number of disadvantaged/marginalized benefiting; impact on poverty (incl. local paid labour); urgency of needs; social cohesion benefits; adaptation to climate risks/impacts; village capacity to implement, operate and maintain.

**Method:** pairwise ranking diagram or show-of-hands voting; count votes transparently in front of plenary. **At least one of the women's priorities must be retained** for transmission to the communal level. All priorities (PCRSS-fundable or not) go into the PDC.

## Tâches du facilitateur (checklist)

- [ ] Faire rapporter et afficher tous les outils MARP ; présenter les scénarios climatiques.
- [ ] Conduire en focus groupes : problèmes → vision → activités → priorisation.
- [ ] Consolider en plénière ; rappeler les critères ; faire le classement par paire.
- [ ] Garantir ≥ 1 priorité des femmes ; consigner toutes les priorités dans le PDC.
