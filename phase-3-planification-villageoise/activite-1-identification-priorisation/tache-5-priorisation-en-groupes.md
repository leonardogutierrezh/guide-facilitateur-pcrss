# Tâche 5 — 5.5 Priorisation en groupes

Classer la liste via le **diagramme de classement par paire** ou vote à mains levées. Préparer le diagramme sur Padex (activités en 1ère ligne reproduites en 1ère colonne ; seules les parties non coloriées sont renseignées). Comparer par paire + vote ; noter les scores ; récapituler ; classer.
